Programación Web

La Progrmación Web es un area muy interesante pero apuesto a que no sabes los conceptos más basicos e importantes, ¿Sabias que una paguina web no es lo mismo que un sitio o una aplicación? Bien por aqui te contamos un poco acerca de esto.

Un poco de historia...

La web nació en 1992 cuando Tim Berners Lee tuvo la genial idea de conectar documentos a través de hipervínculos, si bien en un inicio comenzo siendo muy sencilla con el paso de los años se convirtio en lo que cnocemos ahora.

¿Qué es la WEB

La Web es un conjunto de documentos que se conectan a través de enlaces (hipervínculos). Además junto con la web nace HTML, este lenguaje de marcado es el que permite escribir documentos que se conecten entre ellos.

¿Cuál es la diferencia entre página web, sitio web y aplicación web?

Página web: Es un documento HTML que puede tener CSS o JS.

Sitio web: Es un conjunto de páginas web estructuradas dentro de un dominio. Por ejemplo: ed.team/blog, ed.team/cursos, ed.team/premium, etc.

Las páginas y los sitios web solo son informativos. Básicamente es una forma de tener un folleto con la información de tu empresa en internet. Como las clásicas webs de contacto, quienes somos, nuestros servicios, misión, visión, etc.

Aplicación web: Es un software completo que tiene una lógica de negocio (funciones, usuarios, bases de datos, etc). Un software como tal pero que funciona con tecnologías web. Por ejemplo una red social como Facebook o Twitter, plataformas educativas como EDteam, herramientas de fotos y video, etc.
